Cómo Agregar Nuevos Assets

1\. Coloca el archivo en la carpeta correspondiente dentro del repositorio

2\. Haz commit y push de los cambios

3\. Espera unos minutos a que GitHub Pages actualice

4\. Usa la URL: https://flesan.github.io/assets-flesan/\[carpeta]/\[archivo]
